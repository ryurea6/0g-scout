# 0G Airdrop Scout Agent 🔍

> AI-native airdrop discovery platform built on 0G Network

## 🚀 Features

- **On-Chain Analysis** — Scans smart contracts, TVL, whale movements
- **Social Signals** — Tracks Twitter, Discord, GitHub activity
- **AI Scoring** — 1-10 airdrop potential score via 0G Compute
- **Decentralized Storage** — Results stored on 0G Storage
- **Instant Alerts** — Get notified of high-score projects
- **Portfolio Tracker** — Track your airdrop participation

## 🛠️ Tech Stack

- **Frontend:** HTML/CSS/JS (Zero dependency)
- **AI:** 0G Compute (OpenAI-compatible API)
- **Storage:** 0G Storage SDK
- **Chain:** 0G EVM

## 🏆 Built for 0G Zero Cup

This project was built for the [0G Zero Cup](https://0g.ai/arena/zero-cup) hackathon tournament.

## 📦 Deploy

### GitHub Pages
1. Push to GitHub
2. Settings → Pages → Source: `main` branch
3. Access at `https://username.github.io/0g-scout/`

### Custom Domain
1. Add CNAME file with your domain
2. Configure DNS: `scout.jebod.my.id` → GitHub Pages IP
